Aufgabe

Erstelle eine eigenständige und einfache Webanwendung/Web-API, die die Längen- und Breitengrade (Longitude/Latitude) für eine bestimmte IP4-Adresse ausgeben kann. Du kannst dir die UI als eine einzelne Textzeile vorstellen, die beim Abschicken die Geolocation der Eingabe ermittelt und die Ergebnisse zurückgibt.

Test Anleitung


WebApp
  • Navigiere zu /WebApp
  • In das Feld mit dem Platzhalter "IP Adresse" die IP Adresse in der CIDR notation eintragen
  • Auf den Button "IP Adresse Nachschauen" clicken
  • Das Ergebnis wird über den Eingabe Feldern Angezeigt
API
  • URL: /API?id="Ip Adresse in der CIDR Notation"
  • GET Request
  • JSON als Returnwert
